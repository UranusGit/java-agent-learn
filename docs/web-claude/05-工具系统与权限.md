## 1. 痛点：03/04 章的 Agent 只是"会聊天的鹦鹉"

读完前 4 章你的 Agent 已经能流式对话、能中断——但**它一动手能力都没有**：

- 用户问"帮我看看 server.log 里有什么错误"——Agent 只会说"我没法读文件"
- 用户问"运行一下 npm test"——Agent 只会说"我没法执行命令"
- 用户问"改一下 App.tsx 的标题"——Agent 只能口头描述代码，不能写

> **工具系统**是把 LLM 从"问答机器人"变成"Agent"的关键一跳。它涉及三个子问题：
>
> 1. **怎么让模型知道有这些工具**：把工具 schema 注入 system prompt（OpenAI function calling）
> 2. **怎么执行模型决定的调用**：在 Agent Loop 里识别 `tool_use` 块，调度到对应 Tool
> 3. **怎么不让模型乱来**：每个调用前过 PermissionMiddleware，按规则 allow / deny / ask
>
> 这三件事本章一次做完——但每一件都对应后面几章的深化（24 章智能体安全、20 章审批流、14 章结果预算）。

## 2. 设计回顾（来自 01 章设计文档）

- ToolRegistry：注册所有工具，按权限过滤；
- PermissionMiddleware：工具调用前评估规则（deny > ask > allow）；
- 6 模式：default / acceptEdits / plan / bypassPermissions / autoAccept / yolo；
- 3 行为：allow / deny / ask。

### 2.1 Read

新建 `src/main/java/org/demo02/webclaude/tool/builtin/ReadTool.java`：

```java
// 本代码仅作学习材料参考
package org.demo02.webclaude.tool.builtin;

import org.demo02.webclaude.tool.Tool;
import org.demo02.webclaude.tool.ToolContext;
import org.demo02.webclaude.tool.ToolResult;
import org.springframework.stereotype.Component;

import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Map;
import java.util.concurrent.CompletableFuture;

@Component
public class ReadTool implements Tool {

    @Override
    public String name() { return "Read"; }

    @Override
    public String description() { return "Read a file from the workspace."; }

    @Override
    public Kind kind() { return Kind.READ; }

    @Override
    public Map<String, Object> schema() {
        return Map.of(
            "type", "object",
            "properties", Map.of(
                "path", Map.of("type", "string", "description", "absolute path within workspace")
            ),
            "required", java.util.List.of("path")
        );
    }

    @Override
    public CompletableFuture<ToolResult> apply(Map<String, Object> input, ToolContext ctx) {
        return CompletableFuture.supplyAsync(() -> {
            try {
                String relPath = (String) input.get("path");
                Path root = Path.of(ctx.worktreePath());
                Path resolved = root.resolve(relPath).normalize();

                // 防止路径穿越
                if (!resolved.startsWith(root)) {
                    return ToolResult.error("path traversal denied");
                }

                if (!Files.exists(resolved)) {
                    return ToolResult.error("file not found: " + relPath);
                }

                String content = Files.readString(resolved);
                return ToolResult.text(content);
            } catch (Exception e) {
                return ToolResult.error("read failed: " + e.getMessage());
            }
        });
    }
}
```

```java
import org.demo02.webclaude.tool.Tool;
import org.demo02.webclaude.tool.ToolContext;
import org.demo02.webclaude.tool.ToolResult;

import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Map;
import java.util.concurrent.CompletableFuture;

@Component
public class WriteTool implements Tool {

    @Override public String name() { return "Write"; }
    @Override public String description() { return "Write content to a file in the workspace."; }
    @Override public Kind kind() { return Kind.WRITE; }

    @Override
    public Map<String, Object> schema() {
        return Map.of(
            "type", "object",
            "properties", Map.of(
                "path", Map.of("type", "string"),
                "content", Map.of("type", "string")
            ),
            "required", java.util.List.of("path", "content")
        );
    }

    @Override
    public CompletableFuture<ToolResult> apply(Map<String, Object> input, ToolContext ctx) {
        return CompletableFuture.supplyAsync(() -> {
            try {
                Path root = Path.of(ctx.worktreePath());
                Path resolved = root.resolve((String) input.get("path")).normalize();
                if (!resolved.startsWith(root)) return ToolResult.error("path traversal denied");
                Files.createDirectories(resolved.getParent());
                Files.writeString(resolved, (String) input.get("content"));
                return ToolResult.artifact("wrote " + input.get("path"), guessType(resolved.toString()), resolved.toString());
            } catch (Exception e) {
                return ToolResult.error("write failed: " + e.getMessage());
            }
        });
    }

    private String guessType(String path) {
        if (path.endsWith(".md")) return "markdown";
        if (path.endsWith(".svg")) return "svg";
        if (path.endsWith(".html") || path.endsWith(".htm")) return "html";
        return "code";
    }
}
```

```java
package org.demo02.webclaude.tool.builtin;

import org.demo02.webclaude.tool.Tool;
import org.demo02.webclaude.tool.ToolContext;
import org.demo02.webclaude.tool.ToolResult;
import org.springframework.stereotype.Component;

import java.util.Map;
import java.util.concurrent.CompletableFuture;

@Component
public class BashTool implements Tool {

    @Override public String name() { return "Bash"; }
    @Override public String description() { return "Run a shell command in the workspace."; }
    @Override public Kind kind() { return Kind.EXEC; }

    @Override
    public Map<String, Object> schema() {
        return Map.of(
            "type", "object",
            "properties", Map.of(
                "command", Map.of("type", "string")
            ),
            "required", java.util.List.of("command")
        );
    }

    @Override
    public CompletableFuture<ToolResult> apply(Map<String, Object> input, ToolContext ctx) {
        return CompletableFuture.supplyAsync(() -> {
            try {
                // 注意：本章简化版直接在本机执行（仅开发环境！）
                // 06 章会改为在 Docker 沙箱内执行
                ProcessBuilder pb = new ProcessBuilder("bash", "-c", (String) input.get("command"))
                    .directory(java.nio.file.Path.of(ctx.worktreePath()).toFile());
                pb.redirectErrorStream(true);
                Process p = pb.start();
                boolean finished = p.waitFor(30, java.util.concurrent.TimeUnit.SECONDS);
                if (!finished) {
                    p.destroyForcibly();
                    return ToolResult.error("timeout after 30s");
                }
                String out = new String(p.getInputStream().readAllBytes());
                if (p.exitValue() != 0) {
                    return new ToolResult(out, true, false, null, null);
                }
                return ToolResult.text(out);
            } catch (Exception e) {
                return ToolResult.error("exec failed: " + e.getMessage());
            }
        });
    }
}
```

```java
public class EditTool implements Tool {

    @Override public String description() { return "Replace a substring in a file."; }
    @Override public Kind kind() { return Kind.WRITE; }

    @Override
    public Map<String, Object> schema() {
        return Map.of(
            "type", "object",
            "properties", Map.of(
                "path", Map.of("type", "string"),
                "old", Map.of("type", "string"),

                String replaced = original.replace(
                    (String) input.get("old"),
                    (String) input.get("new")
                );
                if (replaced.equals(original)) {
                    return ToolResult.error("old string not found");
                }
                Files.writeString(resolved, replaced);
                return ToolResult.artifact("edited " + input.get("path"), "code", resolved.toString());
            } catch (Exception e) {
                return ToolResult.error("edit failed: " + e.getMessage());
            }
        });
    }
}
```

```java
// 本代码仅作学习材料参考
package org.demo02.webclaude.tool.builtin;

import org.demo02.webclaude.tool.Tool;
import org.demo02.webclaude.tool.ToolContext;
import org.demo02.webclaude.tool.ToolResult;
import org.springframework.stereotype.Component;

import java.io.IOException;
import java.nio.file.*;
import java.nio.file.attribute.BasicFileAttributes;
import java.util.ArrayList;
import java.util.List;
import java.util.Map;
import java.util.concurrent.CompletableFuture;
import java.util.regex.Pattern;

@Component
public class GrepTool implements Tool {

    @Override public String name() { return "Grep"; }
    @Override public String description() { return "Search a regex pattern in files under the workspace."; }
    @Override public Kind kind() { return Kind.READ; }

    @Override
    public Map<String, Object> schema() {
        return Map.of(
            "type", "object",
            "properties", Map.of(
                "pattern", Map.of("type", "string"),
                "glob", Map.of("type", "string")
            ),
            "required", java.util.List.of("pattern")
        );
    }

    @Override
    public CompletableFuture<ToolResult> apply(Map<String, Object> input, ToolContext ctx) {
        return CompletableFuture.supplyAsync(() -> {
            try {
                Pattern pat = Pattern.compile((String) input.get("pattern"));
                String glob = (String) input.getOrDefault("glob", "**/*");
                PathMatcher matcher = FileSystems.getDefault().getPathMatcher("glob:" + glob);
                Path root = Path.of(ctx.worktreePath());
                List<String> hits = new ArrayList<>();

                Files.walkFileTree(root, new SimpleFileVisitor<>() {
                    @Override
                    public FileVisitResult visitFile(Path file, BasicFileAttributes attrs) throws IOException {
                        if (!matcher.matches(root.relativize(file))) return FileVisitResult.CONTINUE;
                        if (!Files.isRegularFile(file)) return FileVisitResult.CONTINUE;
                        List<String> lines = Files.readAllLines(file);
                        for (int i = 0; i < lines.size(); i++) {
                            if (pat.matcher(lines.get(i)).find()) {
                                hits.add(root.relativize(file) + ":" + (i + 1) + ": " + lines.get(i));
                                if (hits.size() > 200) return FileVisitResult.TERMINATE;
                            }
                        }
                        return FileVisitResult.CONTINUE;
                    }
                });

                return ToolResult.text(String.join("\n", hits));
            } catch (Exception e) {
                return ToolResult.error("grep failed: " + e.getMessage());
            }
        });
    }
}
```

## 5. Tool Registry

新建 `src/main/java/org/demo02/webclaude/tool/ToolRegistry.java`：

```java
// 本代码仅作学习材料参考
package org.demo02.webclaude.tool;

import org.springframework.stereotype.Component;

import java.util.*;
import java.util.stream.Collectors;

@Component
public class ToolRegistry {

    private final Map<String, Tool> tools = new HashMap<>();

    public ToolRegistry(List<Tool> injected) {
        for (Tool t : injected) {
            tools.put(t.name(), t);
        }
    }

    public Tool get(String name) {
        return tools.get(name);
    }

    public Collection<Tool> all() {
        return tools.values();
    }

    public List<Map<String, Object>> schemasForMode(String mode) {
        return all().stream()
            .filter(t -> visibleInMode(t, mode))
            .map(t -> Map.<String, Object>of(
                "name", t.name(),
                "description", t.description(),
                "input_schema", t.schema()
            ))
            .collect(Collectors.toList());
    }

    private boolean visibleInMode(Tool t, String mode) {
        // plan 模式只允许 READ 工具
        if ("plan".equals(mode)) return t.kind() == Tool.Kind.READ;
        return true;
    }
}
```

## 6. Permission Middleware

### 4.1 规则 DSL 解析

新建 `src/main/java/org/demo02/webclaude/permission/RuleMatcher.java`：

```java
// 本代码仅作学习材料参考
package org.demo02.webclaude.permission;

import java.util.Map;
import java.util.regex.Pattern;

public class RuleMatcher {

    // 支持 "Bash(rm -rf:*)" 这种语法
    public static boolean matches(String toolName, Map<String, Object> input, String pattern) {
        if (pattern.equals("*")) return true;
        if (pattern.equals(toolName)) return true;
        if (pattern.startsWith(toolName + "(") && pattern.endsWith(")")) {
            String content = pattern.substring(toolName.length() + 1, pattern.length() - 1);
            return matchesContent(toolName, input, content);
        }
        return false;
    }

    private static boolean matchesContent(String toolName, Map<String, Object> input, String content) {
        // content 格式：field:value 或 field:glob
        // 简化处理：只支持单字段 + glob
        if (content.equals("*")) return true;
        if (content.contains(":")) {
            String[] parts = content.split(":", 2);
            String field = parts[0];
            String valuePattern = parts[1];
            Object fieldValue = input.getOrDefault(field, "");
            String value = fieldValue == null ? "" : fieldValue.toString();
            return Pattern.matches(globToRegex(valuePattern), value);
        }
        // 无字段 → 在 command/path 等关键字段中查找
        for (String key : new String[]{"command", "path", "url"}) {
            Object v = input.get(key);
            if (v != null && Pattern.matches(globToRegex(content), v.toString())) return true;
        }
        return false;
    }

    private static String globToRegex(String glob) {
        StringBuilder sb = new StringBuilder();
        for (char c : glob.toCharArray()) {
            switch (c) {
                case '*': sb.append(".*"); break;
                case '?': sb.append("."); break;
                case '.': sb.append("\\."); break;
                default: sb.append(c);
            }
        }
        return sb.toString();
    }
}
```

```java
public record PermissionRule(
    String scope,        // platform / tenant / project / user / session / default
    String toolName,     // 工具名，* 表示任意
    String pattern,      // DSL content pattern
    Action action,
    int priority

    public enum Action { ALLOW, DENY, ASK }
}
```

新建 `src/main/java/org/demo02/webclaude/permission/PermissionMiddleware.java`：

```java
// 本代码仅作学习材料参考
package org.demo02.webclaude.permission;

import org.demo02.webclaude.tool.Tool;
import org.springframework.stereotype.Component;

import java.util.*;
import java.util.concurrent.CompletableFuture;

@Component
public class PermissionMiddleware {

    // v1：内置默认规则。v1.3+ 接 DB
    private final List<PermissionRule> rules = List.of(
        // 高危：禁止
        new PermissionRule("default", "Bash", "command:rm -rf *", PermissionRule.Action.DENY, 100),
        new PermissionRule("default", "Bash", "command:sudo *", PermissionRule.Action.DENY, 100),
        new PermissionRule("default", "Bash", "command:mkfs*", PermissionRule.Action.DENY, 100),
        new PermissionRule("default", "Read", "path:**/.ssh/**", PermissionRule.Action.DENY, 100),
        new PermissionRule("default", "Read", "path:.env", PermissionRule.Action.DENY, 100),
        new PermissionRule("default", "Write", "path:**/.ssh/**", PermissionRule.Action.DENY, 100)
    );

    public static record Decision(PermissionRule.Action action, String reason) {}

    public CompletableFuture<Decision> evaluate(String toolName, Map<String, Object> input, String mode) {
        return CompletableFuture.supplyAsync(() -> {
            // 1. 模式先判：bypassPermissions / yolo / autoAccept 直接 allow
            if (Set.of("bypassPermissions", "yolo", "autoAccept").contains(mode)) {
                return new Decision(PermissionRule.Action.ALLOW, "mode bypass");
            }
            if ("plan".equals(mode)) {
                // plan 模式只允许 READ
                // （ToolRegistry 已过滤，这里冗余防护）
            }

            // 2. 评估规则（deny > ask > allow）
            PermissionRule.Action effective = null;
            String reason = "no rule matched";
            for (PermissionRule r : rules) {
                if (!RuleMatcher.matches(toolName, input, r.toolName() + "(" + r.pattern() + ")")) continue;
                if (r.action() == PermissionRule.Action.DENY) {
                    return new Decision(PermissionRule.Action.DENY, "rule: " + r.scope() + "/" + r.pattern());
                }
                if (r.action() == PermissionRule.Action.ASK && effective == null) {
                    effective = PermissionRule.Action.ASK;
                    reason = "rule: " + r.scope() + "/" + r.pattern();
                }
                if (r.action() == PermissionRule.Action.ALLOW && effective == null) {
                    effective = PermissionRule.Action.ALLOW;
                    reason = "rule: " + r.scope() + "/" + r.pattern();
                }
            }

            if (effective != null) return new Decision(effective, reason);

            // 3. 无规则 → 按模式默认
            return switch (mode) {
                case "acceptEdits" -> new Decision(PermissionRule.Action.ALLOW, "mode default-allow");
                case "default" -> {
                    // 写操作默认 ask，读操作默认 allow
                    yield new Decision(PermissionRule.Action.ALLOW, "mode default-allow (v1)");
                }
                default -> new Decision(PermissionRule.Action.ALLOW, "mode fallback");
            };
        });
    }
}
```

```java
public Flux<State> runWithTools(UUID sessionId, List<Message> history, String userInput,

private void runLoopRecursive(UUID sessionId, List<Message> history, String userInput,
                              AbortHandle abort, ToolRegistry tools,
                              PermissionMiddleware perm, ToolContext ctx,
                              FluxSink<State> sink) {
    // 第一步：调模型
    callModel(history, userInput, tools, sink, abort,
        assistantMsg -> {
            // 第二步：检查是否有 tool_use
            var toolUses = assistantMsg.content().stream()
                .filter(b -> b instanceof ContentBlock.ToolUse)
                .map(b -> (ContentBlock.ToolUse) b)
                .toList();

            if (toolUses.isEmpty()) {
                sink.next(/* end_turn state */);
                sink.complete();
                return;
            }

            // 第三步：依次执行工具（v1 串行，v2 可并行）
            executeTools(sessionId, toolUses, tools, perm, ctx, sink, abort,
                toolResults -> {
                    // 把 tool_results 追加到 history，递归下一轮
                    var newHistory = new ArrayList<>(history);
                    newHistory.add(assistantMsg);
                    newHistory.addAll(toolResults);
                    runLoopRecursive(sessionId, newHistory, null, abort, tools, perm, ctx, sink);
                });
        });
}
```

> 注：上面是骨架伪代码。完整实现涉及 Spring AI 的 tool-calling 机制，实际可以更简洁地用 `ChatClient.tools(...)` 让框架处理。下面给出简化版整合。

## 12. 避坑：工具系统常踩的雷

| 雷 | 现象 | 规避 |
|----|------|------|
| 工具调用阻塞 Agent Loop | Bash 跑 30s 整个流式停 | 必须 `CompletableFuture` 异步 |
| 工具结果太大撑爆 context | 一次 grep 10MB 文本 | 14 章 ToolResultBudget 裁剪 |
| 同步调用 `join()` 卡死 | 父线程等待子任务死锁 | 用 reactive 链，不要 join |
| Permission 短路绕过 | 模式判断写在工具内部 | 必须在中间件层，统一拦截 |
| 工具 schema 漂移 | 模型给的参数对不上 | 启动时校验 schema 和方法签名 |
| 工具异常被吞 | 模型看不到错误，反复重试 | 异常要变成 `ToolResult.error` 回给模型 |
| 权限规则只看 name 不看 input | 写 `/etc/passwd` 和写 `/tmp/x` 同权限 | DSL 必须支持参数级匹配 |
| 多 turn 内工具结果不持久 | 下一 turn 模型忘了之前调用结果 | tool_result 必须进 message history |
| bypassPermissions 模式泄漏 | 配错导致所有用户都是 yolo | 模式来自可信源（server config / JWT claim）|
| `--cap-drop=ALL` 但工具需要 | sandbox 内 ping 失败 | 这是预期，警惕（沙箱不是开发环境）|
