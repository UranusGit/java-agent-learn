## 3. 前端 Dockerfile

`docker/frontend.Dockerfile`：

```dockerfile
# 本代码仅作学习材料参考
FROM node:20-alpine AS build
WORKDIR /app
COPY package.json pnpm-lock.yaml* ./
RUN npm i -g pnpm && pnpm install --frozen-lockfile
COPY . .
RUN pnpm build

FROM nginx:1.27-alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY ../docker/nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
```

> **前端构建优化（[21 章 §5](./21-Web前端工程化.md)）**：
> - Vite manualChunks 拆分大依赖（Monaco / mermaid / tanstack）；
> - 路由懒加载 `React.lazy`；
> - gzip + brotli 双压缩（nginx 开启 `gzip_static` 和 `brotli_static`）；
> - 资源指纹命名 + 长缓存（`Cache-Control: max-age=31536000, immutable`）；
> - CSP / SRI 头（[23 章 §7](./23-Web安全与可分享性.md)）。

`docker/nginx.conf`：

```
server {
  listen 80;
  server_name _;

  # gzip
  gzip on;
  gzip_types text/plain text/css application/json application/javascript
             application/xml text/javascript image/svg+xml;
  gzip_min_length 1024;

  # 长缓存（带指纹的资源）
  location ~* \.(?:js|css|woff2?|ttf|otf|png|jpg|jpeg|gif|webp|svg)$ {
    root /usr/share/nginx/html;
    expires 1y;
    add_header Cache-Control "public, max-age=31536000, immutable";
    try_files $uri =404;
  }

  # index.html 不缓存
  location = /index.html {
    root /usr/share/nginx/html;
    add_header Cache-Control "no-cache, no-store, must-revalidate";
  }

  # 安全头（[23 章 §7](./23-Web安全与可分享性.md)）
  add_header X-Frame-Options "DENY" always;
  add_header X-Content-Type-Options "nosniff" always;
  add_header Referrer-Policy "strict-origin-when-cross-origin" always;
  add_header Permissions-Policy "geolocation=(), microphone=(), camera=()" always;
  add_header Content-Security-Policy "default-src 'self'; script-src 'self' 'wasm-unsafe-eval'; style-src 'self' 'unsafe-inline'; img-src 'self' data: blob:; font-src 'self' data:; connect-src 'self' wss:; frame-src 'self' blob:; object-src 'none'; base-uri 'self'; form-action 'self'; frame-ancestors 'none'; upgrade-insecure-requests;" always;

  location / {
    root /usr/share/nginx/html;
    try_files $uri $uri/ /index.html;
  }
  location /api {
    proxy_pass http://backend:8080;
    proxy_read_timeout 300s;
  }
  location /ws {
    proxy_pass http://backend:8080;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "upgrade";
    proxy_read_timeout 86400;
    proxy_send_timeout 86400;
    proxy_buffering off;  # 关键：WS 不缓冲
  }
}
```

## 4. 启动 / 停止

### 4.1 首次启动

```bash
# 1. 构建沙箱镜像（02 章已构建）
cd docker/sandbox && docker build -t web-claude-sandbox:latest . && cd ../..

# 2. 创建网络
docker network create web-claude-net

# 3. 一键起
docker compose up -d

# 4. 初始化 bucket
docker run --rm --network=web-claude_default minio/mc \
  alias set local http://minio:9000 minioadmin minioadmin
docker run --rm --network=web-claude_default minio/mc \
  mb local/web-claude || true
```

### 4.2 日常启停

```bash
docker compose up -d
docker compose down
docker compose logs -f backend
```

### 4.3 完全清理

```bash
docker compose down -v   # 删除 volume
docker network rm web-claude-net
```

## 6. 常见问题

### 6.1 沙箱容器起不来

```
Error: sandbox start failed: pull access denied
```

原因：沙箱镜像没构建。
解决：`cd docker/sandbox && docker build -t web-claude-sandbox:latest .`

### 6.2 backend 报 docker socket 权限

```
denied while trying to connect to the Docker daemon socket
```

原因：backend 容器内访问不了宿主 docker。
解决：compose 文件已挂载 `/var/run/docker.sock`，确认宿主用户在 `docker` 组。

### 6.3 WebSocket 一直断

- nginx.conf 的 `proxy_read_timeout` 必须够长（已设 86400）；
- 检查 `proxy_set_header Upgrade`；
- Cloudflare 默认 100s 断 WS，需要 Pro 以上或绕过。

### 6.4 Flyway 报 "schema differs"

- Entity 字段必须和 SQL 完全对齐；
- 不一致时手动 `mvn flyway:repair` 或 `flyway:clean`（仅开发）。

### 6.5 frontend build 时 pnpm-lock 不匹配

```bash
cd frontend
rm pnpm-lock.yaml
pnpm install
```

### 6.6 沙箱内访问不到外网

```bash
docker network inspect web-claude-net
# 检查是否有 gateway，是否启用了 NAT
```

如果是公司网络代理，沙箱要配置 HTTP_PROXY。

## 8. 监控告警

### 8.1 关键指标（Prometheus）

```
webclaude_sessions_active
webclaude_sessions_total
webclaude_agent_run_duration_seconds
webclaude_tool_calls_total{tool, result}
webclaude_sandbox_active
webclaude_sandbox_spawn_duration_seconds
webclaude_tokens_total{tenant, model, direction}
webclaude_errors_total{type}
```

### 8.2 Grafana 仪表板

- 总览：active sessions / day tokens / error rate；
- 单租户：sessions / tokens / cost / errors；
- 沙箱：active / spawn time / kill rate；
- 任务：active tasks / completion rate / avg feature passes。

### 8.3 告警规则

```yaml
# 本代码仅作学习材料参考
groups:
- name: web-claude
  rules:
  - alert: HighErrorRate
    expr: rate(webclaude_errors_total[5m]) > 0.05
    for: 5m
    annotations:
      summary: "Error rate > 5%"
  - alert: SandboxExhausted
    expr: webclaude_sandbox_active > 80
    for: 2m
    annotations:
      summary: "沙箱数接近上限"
  - alert: TokenBurn
    expr: rate(webclaude_tokens_total[10m]) > 100000
    for: 10m
    annotations:
      summary: "10 分钟内 token 用量 > 100k"
```
