## 0. 认知地图

```
文本 → 单模态（你已经会）
    ↓
多模态：
    ├── 图像（vision）—— 看图说话、OCR、图表分析
    ├── 音频（audio）—— 语音转写、TTS
    ├── 文档（PDF/Word）—— 表格抽取、版面理解
    └── 视频（video）—— 帧抽取、动作识别
    ↓
Spring AI 2.0 抽象：
    ├── Media（图片/音频的统一载体）
    ├── MediaFile（本地文件）
    ├── DocumentReader（PDF/Word 解析）
    └── TranscriptionModel（音频转写）
```

## 1. 多模态模型能力对照

| 能力 | OpenAI | Anthropic | 国产 |
|------|--------|-----------|------|
| 图像理解 | GPT-4o / GPT-4o-mini | Claude Sonnet/Opus | Qwen-VL、GLM-4V |
| 音频转写 | Whisper | 不支持 | Paraformer（通义） |
| 文档理解 | 直接 prompt（PDF→图像） | 直接 prompt | Qwen 长文本 |
| 视频理解 | GPT-4o（有限） | 不支持 | Qwen-VL |
| 图像生成 | DALL-E 3 | 不支持 | 通义万相 |

**Java 工程师要点**：理解 + 推理用 LLM，生成（图像/视频生成）通常调专用 API，Spring AI 当前对生成的封装还不完整。

### 2.6 注意：图像 token 成本

每张图像消耗的 token 与分辨率有关。GPT-4o 估算：

| 分辨率 | 大约 token |
|--------|-----------|
| 512×512 | ~285 |
| 1024×1024 | ~1100 |
| 2048×2048 | ~4400 |

**省钱技巧**：先 resize 到 1024 再传，能省 4 倍成本。

```java
public byte[] resizeIfNeeded(byte[] original, int maxSize) throws IOException {
    BufferedImage img = ImageIO.read(new ByteArrayInputStream(original));
    int w = img.getWidth(), h = img.getHeight();
    if (Math.max(w, h) <= maxSize) return original;
    
    double scale = (double) maxSize / Math.max(w, h);
    BufferedImage scaled = new BufferedImage(
            (int)(w * scale), (int)(h * scale), img.getType());
    Graphics2D g = scaled.createGraphics();
    g.setRenderingHint(RenderingHints.KEY_INTERPOLATION,
            RenderingHints.VALUE_INTERPOLATION_BILINEAR);
    g.drawImage(img, 0, 0, scaled.getWidth(), scaled.getHeight(), null);
    g.dispose();
    
    ByteArrayOutputStream out = new ByteArrayOutputStream();
    ImageIO.write(scaled, "png", out);
    return out.toByteArray();
}
```

### 3.1 三种 PDF 处理策略

| 策略 | 做法 | 适合 |
|------|------|------|
| **文本提取** | 用 PDFBox 提取文本 | 文字版 PDF（非扫描件） |

### 3.4 表格抽取

```java
// org.demo02.multimodal.PdfTableExtractor
// 本代码仅作学习材料参考

@Service
public class PdfTableExtractor {

    private final ChatClient chatClient;

    public List<Table> extractTables(Resource pdf, int page) {
        // 1. 用 PDFBox 把指定页转成图
        byte[] pageImage = renderPageToImage(pdf, page);

        // 2. 让 vision 模型识别表格
        Media media = Media.builder()
                .mimeType(MimeTypeUtils.IMAGE_PNG)
                .data(pageImage)
                .build();

        UserMessage message = UserMessage.builder()
                .text("""
                    识别这页所有表格，输出 JSON 数组：
                    [{
                      "title": "表格标题（没有则 null）",
                      "headers": ["列1", "列2"],
                      "rows": [["单元格", "单元格"]]
                    }]
                    不是表格的内容忽略。
                    """)
                .media(media)
                .build();

        List<Table> tables = List.of(chatClient.prompt()
                .messages(message)
                .call()
                .entity(Table[].class));

        return tables;
    }

    private byte[] renderPageToImage(Resource pdf, int page) {
        try (PDDocument doc = Loader.loadPDF(pdf.getInputStream().readAllBytes())) {
            PDFRenderer renderer = new PDFRenderer(doc);
            BufferedImage img = renderer.renderImageWithDPI(page - 1, 200);
            ByteArrayOutputStream out = new ByteArrayOutputStream();
            ImageIO.write(img, "png", out);
            return out.toByteArray();
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

    public record Table(String title, List<String> headers, List<List<String>> rows) {}
}
```

### 3.5 大型 PDF 的分页并发处理

100 页 PDF 串行处理要几十秒，并发：

```java
@Service
public class ParallelPdfProcessor {

    private final PdfTableExtractor extractor;

    public PdfProcessResult process(Resource pdf, int totalPages) {
        return Flux.range(1, totalPages)
                .flatMap(page -> Mono.fromCallable(() -> {
                    List<Table> tables = extractor.extractTables(pdf, page);
                    return new PageResult(page, tables);
                }))
                .subscribeOn(Schedulers.boundedElastic())
                .collectList()
                .map(PdfProcessResult::new)
                .block();
    }

    public record PageResult(int page, List<Table> tables) {}
    public record PdfProcessResult(List<PageResult> pages) {}
}
```

**注意限流**：并发的 LLM 调用会爆 token quota。

## 4. Word / Excel 处理

### 4.1 Word（.docx）

Spring AI 没原生支持，用 Apache POI：

```xml
<dependency>
    <groupId>org.apache.poi</groupId>
    <artifactId>poi-ooxml</artifactId>
    <version>5.2.5</version>
</dependency>
```

```java
public class DocxReader {

    public List<Document> read(Resource docx) {
        try (XWPFDocument doc = new XWPFDocument(docx.getInputStream())) {
            List<Document> result = new ArrayList<>();
            
            // 提取段落
            for (XWPFParagraph para : doc.getParagraphs()) {
                String text = para.getText();
                if (!text.isBlank()) {
                    result.add(new Document(text, Map.of(
                            "type", "paragraph",
                            "style", para.getStyle()
                    )));
                }
            }
            
            // 提取表格
            for (XWPFTable table : doc.getTables()) {
                String markdown = tableToMarkdown(table);
                result.add(new Document(markdown, Map.of("type", "table")));
            }
            
            return result;
        }
    }

    private String tableToMarkdown(XWPFTable table) {
        StringBuilder sb = new StringBuilder();
        for (XWPFTableRow row : table.getRows()) {
            sb.append("| ");
            for (XWPFTableCell cell : row.getTableCells()) {
                sb.append(cell.getText()).append(" | ");
            }
            sb.append("\n");
        }
        return sb.toString();
    }
}
```

### 4.2 Excel

public class ExcelReader {

    public List<Document> read(Resource xlsx) {
        try (Workbook wb = WorkbookFactory.create(xlsx.getInputStream())) {
            List<Document> result = new ArrayList<>();
            DataFormatter fmt = new DataFormatter();
            
            for (Sheet sheet : wb) {
                StringBuilder sb = new StringBuilder();
                for (Row row : sheet) {
                    sb.append("| ");
                    for (Cell cell : row) {
                        sb.append(fmt.formatCellValue(cell)).append(" | ");
                    }
                    sb.append("\n");
                }
                result.add(new Document(sb.toString(), Map.of(
                        "type", "spreadsheet",
                        "sheet", sheet.getSheetName()
                )));
            }
            return result;
        }
    }
}

public List<Anomaly> detectAnomalies(Resource video) {
    List<byte[]> frames = frameExtractor.extractFrames(video, 1);
    List<Anomaly> anomalies = new ArrayList<>();

    for (int i = 0; i < frames.size(); i++) {
        Media media = Media.builder()
                .mimeType(MimeTypeUtils.IMAGE_PNG)
                .data(frames.get(i))
                .build();

        AnomalyCheck check = chatClient.prompt()
                .messages(UserMessage.builder()
                        .text("""
                            判断画面是否有异常（打架、跌倒、火情、入侵）。

                        .media(media)
                        .build())
                .call()
                .entity(AnomalyCheck.class);

        if (check.hasAnomaly() && check.confidence() > 0.7) {
            anomalies.add(new Anomaly(i, check.type(), check.confidence()));
        }
    }
    return anomalies;
}

@Service
public class MultimodalIngestionService {

    public void ingest(Resource pdf) {
        List<Document> docs = pdfReader.read(pdf);
        
        for (Document doc : docs) {
            // 1. 纯文本部分：直接嵌入
            if ("paragraph".equals(doc.getMetadata().get("type"))) {
                vectorStore.add(List.of(doc));

                Document enriched = new Document(
                        description + "\n\n[图片描述]",
                        Map.of("original_image", doc.getMetadata().get("image_path"))
                );
                vectorStore.add(List.of(enriched));
            }
        }
    }
}

## 10. 实战避坑

### 10.1 "LLM 说看不到图片"

**原因**：用的模型不支持 vision（如 GPT-3.5、Claude Haiku 早期版本）。

**解决**：用 GPT-4o / Claude Sonnet 4+ / Qwen-VL 等明确支持 vision 的模型。

### 10.2 "上传 PDF 报 file too large"

**原因**：模型 API 限制单文件 20MB。

**解决**：

- 大 PDF 先按页切。
- 扫描件先压缩或转单页图。

### 10.3 "Whisper 转写中文乱码"

**原因**：没指定 language。

**解决**：`.language("zh")`。

### 10.4 "图片 base64 编码后请求体太大"

**原因**：很多网关对请求体有限制（如 10MB）。

**解决**：

- 改用 URL 模式（让模型自己下载）。
- 或先上传到对象存储，给 URL。

### 10.5 "流式输出图像分析时丢内容"

**原因**：流式 chunk 较小，前端拼接处理不当。

**解决**：见 26 篇 Reactor 流式深度。

### 10.6 "多模态调用慢得离谱"

**原因**：高分辨率图 + 大 PDF + 串行处理。

**解决**：

- 降分辨率。
- 切片并行（控制并发度）。
- 用 cache。
